学习Python的第二步：Pycharm的安装教程（超详细）
发表于 2020-07-10| 更新于 2020-07-10
阅读量:|评论数:

学会了安装Python之后，安装一款顺手好用的IDE自然是必不可少的，正如标题所说，我选择了Pycharm，它与IDEA同属JetBrains公司，界面相似，功能齐全，是个不错的选择。

一、Pycharm的安装

（1）首先打开Pycharm的官网（https://www.jetbrains.com/pycharm/download/），在左侧可以看到最新版本以及发布时间，右侧提供了专业版以及社区版的下载，我们点击专业版的Download下载按钮。

（2）点击之后会跳转到感谢下载页面，并弹出下载框，下载即可。

值得一提的是，这软件下载速度可比Python快多了，一两分钟就下载完毕。

（3）下载完毕之后运行安装程序，第一页直接next

（4）选择安装路径，软件占用内存较大，建议在其他盘新建一个pycharm的文件夹，并且注意路径不要带有中文，点击next。

（5）这一步全部勾选，单击Next
①：创建桌面快捷方式（64位）
②：更新上下文菜单，添加打开文件夹作为项目
③：创建关联，关联.py文件，以pycharm打开
④：更新路径变量(需要重新启动)，将启动器目录添加到路径中

（6）这一步直接点击Install即可

（7）正在安装，稍等即可安装完成

（8）安装完成，选择第二个稍后重启，点击Finish完成安装

二、配置Pycharm

（1）在桌面上找到刚刚安装好的Pycharm并打开，选择不导入设置，点击OK

（2）来到主题设置页面，根据自己喜好选择即可，然后点击右下角Next，或者也可以直接左下角Skip和右上角X来使用默认设置

（3）来到Download featured plugins页面，直接start using pycharm就好

（4）进入激活页面，可以选择是激活还是试用30天，这里不再详述。
